# Synced Momentum - Coach Platform

Web application for fitness coaches to manage clients, review check-ins, assign programmes and meal plans.

**Related Project:** [Synced Momentum iOS App](https://github.com/lawsonsenterprises/fitness-app-ios)

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel

## Development
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
